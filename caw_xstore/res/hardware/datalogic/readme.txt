This folder contains files and information that can be used to program and install Datalogic Gryphon and Magellan scanners for use with Xstore.

If using a Gryphon scanner, please see the setup instructions document in the gryphon-4xx0 folder for installation and configuration details.
If using a Magellan scanner, please see the setup instructions document in the magellan-1100i folder for installation and configuration details.
